TIL's

  1. .(dot command)

    In vim, the dot command repeats last change. If you have searched for a word and want to change all occurences of it, in normal mode type cgn[word], now pressing the dot will change the next occurences on each press.

  2. ; and ,

    In vim, if you want to repeat the search for a character done using t, f, T or F, pressing ; does it the motion's direction and , in the opposite direction.

  3. gcc --save-temps source.c

    Command to see intermediate files of a C source file compilation

  4. fc

    Unix utility command to execute to edit the last executed command in the default editor, very helpful while getting SQL queries right.

  5. entr

    Run arbitrary commands when files change
